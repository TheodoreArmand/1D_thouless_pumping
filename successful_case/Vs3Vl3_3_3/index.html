2. Continuum Hamiltonian Actually Integrated

\[ H(t) = \sum_{a=1}^{N} \left[ -\frac{\hbar^2}{2m} \frac{\partial^2}{\partial x_a^2} - V_s\cos\!\left(\frac{4\pi x_a}{a}\right) - V_l\cos\!\left(\frac{2\pi x_a}{a}+\phi(t)\right) \right]. \]
当前案例 \(N=1\), \(V_s=V_l=3E_R\). 这里没有 harmonic trap, 没有 imaginary-time initialization.
\[ \phi(t)=\Phi(s),\qquad s=\frac{t}{T},\qquad \Phi \text{ is linearly interpolated from the CSV schedule.} \]
RK4 的时间网格是固定 dt；相位不是 \(\phi=2\pi t/T\)，而是由 gap-adaptive CSV 查询得到。
\[ |\psi(z)\rangle = \sum_{k=1}^{16} u_k \exp\!\left[-(A_k+B_k)x^2+2B_kR_kx-B_kR_k^2\right], \qquad z=(u,A,B,R). \]
这是本 N=1 案例的 ECG 可视化形式；notebook 用同一个形式重建 density。
\[ i\,C_{\alpha\beta}\dot z_\beta=g_\alpha,\qquad C_{\alpha\beta}= \langle\partial_\alpha\psi|\partial_\beta\psi\rangle,\qquad g_\alpha= \langle\partial_\alpha\psi|H|\psi\rangle . \]
这就是 real-time Dirac-Frenkel TDVP 的线性方程；当前运行用正则化 SVD solve。

3. Rice-Mele Reference Model

连续双阱最低两条能带可以投影成 Rice-Mele 两站点模型。这里的 \(J_1\) 是 intracell hopping， \(J_2\) 是 intercell hopping，\(\Delta=\epsilon_A-\epsilon_B\) 是子晶格能量差。

\[ H_{\mathrm{RM}}(k,\phi) = \begin{pmatrix} \Delta(\phi)/2 & -\left[J_1(\phi)+J_2(\phi)e^{-ikd_l}\right] \\ -\left[J_1(\phi)+J_2(\phi)e^{ikd_l}\right] & -\Delta(\phi)/2 \end{pmatrix}. \]
\[ g(k,\phi) = 2\sqrt{ \left[\frac{\Delta(\phi)}{2}\right]^2 +J_1(\phi)^2+J_2(\phi)^2 +2J_1(\phi)J_2(\phi)\cos(kd_l) }, \] \[ g_{12}(\phi)=\min_k\{E_2(k,\phi)-E_1(k,\phi)\}. \]
main.cpp schedule grid: \(g_{\min}=0.562417E_R\)
RM grid file: \(g_{12}^{\min}=0.473767E_R\)
RM grid file: \(g_{23}^{\min}=0.585910E_R\)
two-band isolation: \(g_{23}^{\min}/g_{12}^{\min}=1.2367\)
这个页面描述的是历史成功 run：它用 half-depth-derived schedule （\(g_{\min}=0.562417E_R\)）驱动 full-depth Hamiltonian。当前 canonical 协议改为 full-depth-derived schedule，与 grid_rm_curves_Vs3Vl3.npz 的 \(g_{12}^{\min}=0.473767E_R\) 一致。旧 schedule 和结果只用于复现，不应用来代表新协议。
